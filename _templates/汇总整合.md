# 📊 全局汇总中心

## 1. 所有未完成待办

```d
TASK
WHERE !completed
SORT file.name ASC
```

## 2. 所有已完成任务（复盘用）

```d
TASK
WHERE completed
SORT file.name ASC
```

## 3. 最近7天新建笔记

```d
LIST
WHERE file.ctime >= date(today) - dur(7 days)
SORT file.ctime DESC
```

## 4. 所有日期日记自动汇总

```d
LIST link(file.name) 
WHERE regexmatch("\\d{4}-\\d{2}-\\d{2}", file.name) 
SORT file.name DESC 
LIMIT 30
```
### 🔹 今日笔记 
```d
LIST link(file.name) 
WHERE file.name = date(today).toString 
SORT file.name DESC
```
### 🔹 最近3天笔记
```d
LIST link(file.name) 
WHERE file.ctime >= date(today) - dur(3 days) 
SORT file.ctime DESC
```
### 🔹 最近7天笔记
```d
LIST link(file.name) 
WHERE file.ctime >= date(today) - dur(7 days) 
SORT file.ctime DESC
```

### 🔹 本周所有笔记 
```d
dataview LIST link(file.name) 
WHERE file.ctime >= date(today).startof("week")

```
### 🔹 本月所有笔记
```d
LIST link(file.name) 
WHERE file.ctime >= date(today).startof("month") 
SORT file.ctime DESC
```

## 5. 库内笔记总数统计

```
TABLE 
length(filter(file.tasks, t => !t.completed)) AS 未完成任务,
length(filter(file.tasks, t => t.completed)) AS 已完成任务,
length(file) AS 总笔记数
```

## 6. 只汇总某个文件夹（改引号里名字就行）

把 `"模板"` 改成你自己的文件夹名

```
LIST
FROM "模板"
SORT file.name ASC
```
